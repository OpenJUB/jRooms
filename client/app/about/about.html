What's up?

This is an updated version of the yearly room allocation system for Jacobs University. This project is written in a modern development environment (Node.js + Angular), which allows for fast and modular improvements.

It uses OpenJUB, that is running on a secure USG server, and has no direct access to your CampusNET credentials or any other sensitive data.

This project is open-source! You can look through the code on Github (as a part of OpenJUB organization).
